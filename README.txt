### COMERCIANTE

Palomma permite a los comercios ahorrar en costos de procesamiento y aumentar conversión con pagos bancarios seguros de un solo clic.

**Beneficios**

**AHORRO:** Disminuye entre 70 y 80% tus costos de procesamiento de pagos

**RECURRENCIA:** Ofrece la posibilidad de pagos recurrentes

**ESTABILIDAD:** Mejora la estabilidad de tus medios de pago

**EXPERIENCIA DE USUARIO:** Ofrece una super experiencia de pago a tus usuarios; simple, segura, y a un clic.

== Installation ==

= Desde Dashboard ( WordPress admin ) =
* plugins -> Agregar Nuevo
* buscar 'palomma'
* click en Instalar Ahora y luego en Activo.

= Usando FTP o similar =
* Unzip el archivo "palomma" y
* Carga el folder "palomma" al directorio "/wp-content/plugins/".
* Activa el plugin a través dde la sección "Plugins" en el menú de WordPress
